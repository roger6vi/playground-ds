# App consumidora del Design System (placeholder)

Aplicación React que demostrará y validará el consumo del Playground Design System.

Estructura inicial:
- `src/main.jsx`: punto de entrada de React.
- `src/App.jsx`: shell de la aplicación.
- `src/pages/`: vistas o páginas de documentación.
- `src/components/`: componentes específicos de la app (no del DS).

Instala dependencias y configura Vite/CRA u otra herramienta cuando se empiece a implementar.
